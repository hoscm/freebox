# freebox
freeBox - Open Source version of hsBox

freeBox (v1.3 Beta)
freeBox は、USB起動のLiveモードで動作する軽量・堅牢なシステム環境 hsBox のコンセプトを継承したオープンソース・プロジェクトです。

🚀 特徴
Liveモード最適化: USBメディアから直接起動し、OSのシステム領域を汚さない運用が可能です。

独自のタスク管理: /etc/crontab を直接編集せず、専用の反映スクリプトを用いることで、Live環境でも確実なタスク実行を実現します。

軽量・高速: 余分なサービスを削ぎ落とし、特定のタスク（サーバー運用やエッジデバイス）に特化しています。


⚠️ 注意事項
本システムはLiveモードで動作するため、/etc/fstab によるマウント設定は無効です。自動マウント機能については次期バージョン（v1.4相当）での強化を予定しています。
